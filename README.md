# xg-azure
Deployment template to deploy Sophos XG firewall to Azure

Deploying
=========

This repository provides an example template for High Availability deployments which will create multiple XG firewalls and an Azure load balancer.

Configuration sync between the XG nodes can be done by using SCFM. Please reach out to your sales or channel representative to learn more about this Sophos product.

[![Deploy to Azure](https://azuredeploy.net/deploybutton.png)](https://portal.azure.com/#create/Microsoft.Template/uri/https%3A%2F%2Fraw.githubusercontent.com%2Fiaasteamtemplates%2FXGHATest%2FAvSet%2FinboundHa.json)
